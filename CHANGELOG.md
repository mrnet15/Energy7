# Changelog

## v1.0 — first release

Energy 7's first stable release: a complete automatic-DJ app for Windows.

**Mixing**
- Beat-grid-locked crossfades (downbeat entry, whole-bar fades).
- Three mix modes: Beat-aligned (keep tempo), Tempo-matched (beat-lock, with
  half/double-time handling), and EQ bass-swap (club-style clean blend).
- Optional Filter + echo transition FX, layerable on any mode.
- Loudness normalization (LUFS) for a consistent set.
- Long-intro detection so tracks start at the drop.

**Sequencing**
- Auto-Order: harmonic (Camelot key) + tempo sequencing for the smoothest set.
- Lock Start to pin a chosen opener.
- Manual BPM / Key overrides (double-click a cell).

**Playback & UI**
- Modern dark interface with a neon logo.
- Live transport: play/pause/stop, click-to-scrub, ±10s skip, next/previous track.
- Trippy kaleidoscope visualizer.
- Drag-and-drop to add files.

**Output**
- Export the mix as a single 320 kbps MP3.
- Save a timestamped tracklist and a .cue sheet.
- Save/load .m3u playlists and .bmx projects.
- Persistent analysis cache so re-adding tracks is instant.
